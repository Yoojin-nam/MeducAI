# Archived Code

## 개요

이 폴더에는 현재 사용하지 않는 legacy 코드와 백업 파일들이 보관되어 있습니다.

---

## 폴더 구조

### 주요 하위 폴더

- **`src_old/`**: 이전 버전의 소스 코드 (최초 파이프라인 구현)
- **`src_2/`**: 중간 버전의 소스 코드 (v2 구조 전환기)
- **`refer/`**: 참고용 코드 예제
- **`notebooks/legacy/`**: 과거 탐색 및 분석 노트북

### 백업 파일들

**최신 백업 2개만 보관** (2026-01-07 정리 완료):
- `01_generate_json.py.BAK_20251218_002439`: 최종 백업 (2025-12-18 00:24)
- `01_generate_json.py.BAK_20251218_001514`: 이전 백업 (2025-12-18 00:15)

이전의 40+ 개 백업 파일은 삭제되었습니다.

---

## 현재 사용 중인 코드 위치

현재 사용 중인 코드는 다음 위치에 있습니다:

- **`3_Code/src/`**: 현재 사용 중인 소스 코드
- **`3_Code/Scripts/`**: 실행 스크립트들
- **`3_Code/notebooks/`**: Jupyter 노트북들

## 정리 규칙

### 보관 목적

1. **히스토리 보존**: 과거 구현의 변화 과정 추적
2. **참고 자료**: 특정 기능의 이전 구현 방식 참고
3. **롤백 대비**: 필요시 특정 버전으로 복원

### 사용 주의사항

- ❌ **현재 파이프라인에서 직접 사용하지 않음**
- ✅ **참고 용도로만 사용**
- ✅ **현재 코드 수정 시 참고 가능**

## 주요 Legacy 코드 설명

### `src_old/`

**목적**: 최초 파이프라인 구현 버전 보존

이전 버전의 파이프라인 코드들:
- `01_generate_json.py`: 이전 버전의 JSON 생성 스크립트
- `run_arm_full.py`: 이전 버전의 ARM 실행 스크립트
- `03_generate_images.py`, `04_export_anki.py` 등
- 기타 초기 구현들

**참고 용도**: 특정 기능의 이전 구현 방식 확인 시 유용

### `src_2/`

**목적**: v2 구조 전환기 코드 보존

v2 구조의 중간 버전 코드들:
- 모듈화 및 구조 개선 과정의 중간 산출물
- `src_old/`와 현재 `src/` 사이의 변화 추적 가능

**참고 용도**: 리팩토링 히스토리 추적

### `refer/`

**목적**: 참고용 코드 예제 및 프로토타입

참고용 코드 예제들:
- `01_generate_json_refer.py`
- `02_postprocess_refer.py`
- `03_generate_refer.py`
- `04_export_ank_refer.py`

**참고 용도**: 특정 기능 구현 시 참고 코드

### `notebooks/legacy/`

**목적**: 과거 탐색 및 분석 노트북 보존

이전 버전의 분석 및 탐색 노트북들
