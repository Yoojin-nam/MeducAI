## 1. AppSheet 시스템 사용법

### 1.1 시스템 접속

#### 접속 환경
- **모바일**: iOS/Android (App Store/Google Play에서 'AppSheet' 검색)
- **컴퓨터**: 웹 브라우저 (Chrome, Safari, Edge 권장)
- **계정**: 제공받은 Google 계정으로 로그인

#### 접속 방법
1. 제공받은 AppSheet 링크 클릭 또는 앱 실행
2. Google 계정 로그인 (승인된 계정만 접근 가능)
3. 'MeducAI QA' 앱 선택

### 1.2 화면 구성

#### 하단 탭
- **TODO 탭**: 평가 대기 중인 카드 목록
  - 아직 평가하지 않은 카드 표시
  - 카드 번호 클릭하여 평가 시작
  
- **DONE 탭**: 평가 완료한 카드 목록
  - 완료된 카드 확인 및 재검토 가능

### 1.3 평가 시작하기

1. **TODO 탭**에서 카드 번호 클릭
2. 카드 내용 충분히 검토
3. **🚀 로켓 버튼** 클릭 → **이 시점부터 시간 측정 시작**
4. 평가 진행

**중요**: 로켓 버튼을 누르기 전에 카드를 충분히 읽고 평가 준비를 완료하세요!

### 1.4 필수 vs 선택 항목

| 구분 | 항목 |
| --- | --- |
| **필수** | 문항 평가, Diagram 이미지 평가, AI 평가 품질, Regen 수용 여부 |
| **선택** | Realistic 이미지 평가 (있는 경우), 코멘트, 메모 |

**Tip**: 필수 항목을 입력하지 않으면 다음 단계로 진행할 수 없습니다.

## 2. 평가 개요

### 2.1 평가 대상

MeducAI 프로젝트의 FINAL QA(최종 품질 보증) 평가는 다음을 대상으로 합니다:

| 평가 대상 | 설명 |
| --- | --- |
| **카드 콘텐츠** | Anki 카드의 텍스트 내용 (Q1: 단답형, Q2: MCQ) |
| **Diagram 이미지** | 카드에 포함된 의학 영상 (CT, MRI, US, X-ray 등) |
| **Realistic 이미지** | AI가 생성한 사실적 의료 영상 (약 300장, 일부만 해당) |
| **인포그래픽** | AI가 생성한 Table/Infographic |

**중요**: FINAL QA의 주요 평가 대상은 **카드 콘텐츠**와 **이미지**입니다.

### 2.2 평가 단계 (3단계)

| 단계 | 이름 | 설명 | 소요 시간 |
| --- | --- | --- | --- |
| **Stage 1** | Pre 평가 | AI 결과 보기 전 순수 평가 | 약 1-1.5분 |
| **Stage 2** | Post 평가 | AI 결과 확인 후 재평가 | 약 1-1.5분 |
| **Stage 3** | Final Assessment | AI 품질 평가 + Regen 검토 | 약 1분 |

**카드당 총 예상 시간**: 약 2-3분 (정확성이 속도보다 우선!)

### 2.3 평가 통계

| 항목 | 수량 |
| --- | --- |
| **전체 카드 수** | 약 6,000장 |
| **Realistic 이미지** | 약 300장 (전공의는 일부만) |
| **Regen 카드** | 약 400장 (~6.7%) |

## 3. 평가 절차

### 3.1 Stage 1: Pre 평가 (AI 결과 보기 전)

Stage 1은 AI의 평가를 보지 않은 상태에서 진행하는 **독립적이고 순수한 평가**입니다.

#### 평가 흐름

```
1. 카드 콘텐츠 확인
   - Q1 (단답형 주관식) 검토
   - Q2 (MCQ) 검토
   ↓
2. 카드 평가 항목 입력 (필수)
   - Blocking Error (Yes/No)
   - Technical Accuracy (0, 0.5, 1.0)
   - Educational Quality (1-5점)
   - Evidence Comment (조건부 필수)
   ↓
3. 이미지 확인 및 평가 (필수)
   - 카드에 포함된 이미지 직접 확인
   - Image Blocking Error (Yes/No)
   - Image Anatomical Accuracy (0, 0.5, 1.0)
   - Image Quality (1-5점)
   - Image-Text Consistency (0, 0.5, 1.0)
   ↓
4. Realistic 이미지 평가 (선택, 있는 경우만)
   ↓
5. "Submit Pre-S5 Rating" 버튼 클릭
```

**중요**: Pre 평가는 제출 후 변경할 수 없습니다 (immutable).

### 3.2 Stage 2: Post 평가 (AI 결과 확인 후)

Stage 2에서는 AI의 평가 결과를 참고하여 재평가합니다.

#### 평가 흐름

```
1. "Reveal S5 Results" 버튼 클릭
   - AI 자가 평가 점수 확인
   - 이미지 품질 평가 결과 확인
   - 개선 제안 및 근거 검토
   - 재생성 콘텐츠 여부 확인
   ↓
2. AI 결과 검토
   - AI가 발견한 문제점 확인
   - 개선 제안의 합리성 평가
   ↓
3. Pre 평가와 동일한 기준으로 재평가
   - 카드 평가 항목 입력 (동일 기준)
   - 이미지 평가 항목 입력 (동일 기준)
   ↓
4. 점수 변경 시 이유 선택 (필수)
   ↓
5. "Submit Post-S5 Rating" 버튼 클릭
```

**중요**: Pre와 Post는 **동일한 평가 기준** 사용. AI 결과를 참고할 뿐, 기준 자체는 변하지 않습니다.

#### 점수 변경 이유 (필수)

점수를 변경하는 경우 다음 중 선택:

| 변경 이유 | 설명 |
| --- | --- |
| **AI가 놓친 문제점 발견** | AI가 지적하지 못한 문제를 발견 |
| **AI의 의견에 동의** | AI의 평가가 더 타당하다고 판단 |
| **새로운 관점 획득** | AI 분석을 통해 새로운 시각 확보 |
| **초기 평가 실수** | Stage 1에서 실수나 오판 |
| **기타** | 위 항목에 해당하지 않는 경우 |

### 3.3 Stage 3: Final Assessment

Stage 3은 **AI 평가 품질 검증** 및 **Regen 카드 처리** 단계입니다.

#### 평가 흐름

```
1. AI 자가 평가 신뢰도 평가 (필수)
   - 1-5점 척도로 AI 평가의 신뢰도 평가
   - 신뢰도 평가 근거 작성 (선택)
   ↓
2. Regen 카드 처리 (재생성된 경우만)
   - 원본 vs 재생성 버전 비교
   - Accept/Reject 결정 (필수)
   - AI 수정안 품질 평가 1-5점 (필수)
   - 수용/거부 근거 작성 (선택)
   ↓
3. 후속 조치 필요 플래그 (선택)
   - 치명적 문제 발견 시 체크
   - 플래그 메모 작성 (선택)
   ↓
4. "Submit Final Assessment" 버튼 클릭
```

## 4. 평가 항목 및 점수 기준

### 4.1 카드 평가 항목

#### 4.1.1 Blocking Error (치명적 오류) - 필수

**질문**: 이 카드에 사용을 막아야 할 정도의 치명적 오류가 있나요?

| 선택 | 설명 | 예시 |
| --- | --- | --- |
| **Yes** | 치명적 오류 있음 | 정답 오류, 해부학적 오류, 중대한 의학적 오류 |
| **No** | 치명적 오류 없음 | 사소한 오타, 표현 개선 필요 등 |

**판단 기준**:

**Yes (치명적 오류)**:
- 정답이 명백히 잘못됨
- 해부학적 구조나 병리학적 내용이 틀림
- 환자 안전에 영향을 줄 수 있는 오류
- 전공의 교육에 심각한 혼란을 줄 수 있음

**No (치명적 오류 아님)**:
- 사소한 오타나 문법 오류
- 표현이 다소 모호하나 의미 파악 가능
- 교육적 가치는 있으나 개선 여지 있음

**중요**: Blocking Error가 "Yes"인 경우, **반드시 Evidence Comment에 구체적인 오류 내용을 작성**해야 합니다.

#### 4.1.2 Technical Accuracy (의학적 정확성) - 필수

**질문**: 카드의 의학적 내용이 얼마나 정확한가요?

| 점수 | 설명 | 예시 |
| --- | --- | --- |
| **1.0** | 완전히 정확 | 의학 교과서와 일치, 최신 가이드라인 반영 |
| **0.5** | 대체로 정확, 일부 불완전 | 내용은 맞으나 설명이 불충분하거나 모호함 |
| **0.0** | 부정확 | 의학적 오류 포함, 잘못된 정보 |

**평가 기준**:

**1.0점 (완전히 정확)**:
- 질문, 선택지, 정답, 해설 모두 의학적으로 정확
- 최신 의학 지식 및 가이드라인과 일치
- 해부학, 병리학, 영상 소견 모두 정확
- 전공의 교육에 바로 사용 가능

**0.5점 (대체로 정확, 일부 불완전)**:
- 주요 내용은 정확하나 부분적 불완전성 존재
- 설명이 다소 모호하거나 불충분
- 최신 정보가 아니지만 크게 문제되지 않음
- 맥락상 의미는 파악 가능

**0.0점 (부정확)**:
- 의학적 오류가 명확히 존재
- 해부학적 구조나 병리 소견이 틀림
- 정답이 잘못되었거나 선택지에 오류
- 전공의 교육에 혼란을 줄 수 있음

**Tip**: 애매한 경우 0.5점을 선택하고, Evidence Comment에 어떤 부분이 불완전한지 기록하세요.

#### 4.1.3 Educational Quality (교육적 품질) - 필수

**질문**: 이 카드가 전공의 교육에 얼마나 유용한가요?

| 점수 | 설명 |
| --- | --- |
| **5점** | 매우 우수 - 즉시 교육에 사용 가능 |
| **4점** | 우수 - 대체로 좋으나 소소한 개선 여지 |
| **3점** | 보통 - 사용 가능하나 개선 권장 |
| **2점** | 미흡 - 상당한 수정 필요 |
| **1점** | 매우 미흡 - 사용 부적절 |

**평가 기준**:

**5점 (매우 우수)**:
- 학습 목표가 명확하고 적절함
- 질문과 해설이 교육적으로 우수
- 전공의 수준에 딱 맞는 난이도
- 임상 실무에 직접 도움됨
- 이미지와 텍스트가 완벽히 조화

**4점 (우수)**:
- 교육적 가치가 높음
- 일부 개선 여지 있으나 크게 문제없음
- 전공의 학습에 충분히 유용
- 전반적으로 만족스러움

**3점 (보통)**:
- 평균적인 교육 자료 수준
- 사용 가능하나 개선 권장
- 일부 내용이 불명확하거나 불충분
- 난이도가 다소 높거나 낮음

**2점 (미흡)**:
- 교육적 가치가 낮음
- 상당한 수정이 필요
- 학습 목표가 불명확
- 설명이 매우 부족하거나 혼란스러움

**1점 (매우 미흡)**:
- 교육 자료로 부적절
- 전공의 학습에 도움 안 됨
- 내용이 매우 혼란스럽거나 오해 소지
- 사용 불가능

**Tip**: 평가 시 "내가 전공의 1년차라면 이 카드가 학습에 도움이 될까?"를 기준으로 판단하세요.

#### 4.1.4 Evidence Comment (근거 코멘트) - 조건부 필수

**질문**: 평가의 근거나 구체적인 문제점을 설명해주세요.

**작성 조건** (다음 중 하나라도 해당 시 **필수**):

1. **Blocking Error = Yes**: 치명적 오류의 구체적 내용 필수 작성
2. **Technical Accuracy = 0.0 또는 0.5**: 어떤 부분이 부정확하거나 불완전한지 설명
3. **Educational Quality ≤ 2점**: 교육적 품질이 낮은 이유 설명

**작성 가이드**:

- **구체적으로**: "오류 있음" 대신 "정답이 B인데 C로 표시됨"
- **간결하게**: 핵심만 1-2문장으로
- **건설적으로**: 비판보다는 개선 방향 제시

**예시**:

✅ **좋은 예시**:
- "Q2 정답이 'Pancreatic adenocarcinoma'인데 'Chronic pancreatitis'로 표시됨"
- "IVC의 위치가 해부학적으로 부정확함 (좌측으로 표시)"
- "해설에서 BI-RADS 카테고리 설명이 누락됨"

❌ **나쁜 예시**:
- "틀렸음"
- "내용이 이상함"
- "문제 있음"

### 4.2 이미지 평가 항목

#### 4.2.1 Image Blocking Error (이미지 치명적 오류) - 필수

**질문**: 이미지에 사용을 막아야 할 정도의 치명적 오류가 있나요?

| 선택 | 설명 | 예시 |
| --- | --- | --- |
| **Yes** | 치명적 오류 있음 | 잘못된 이미지, 해부학적 오류, 판독 불가 |
| **No** | 치명적 오류 없음 | 이미지 품질은 괜찮음 |

**Yes (치명적 오류)**:
- 완전히 다른 이미지가 첨부됨 (텍스트와 불일치)
- 해부학적 구조가 명백히 틀림
- 이미지가 너무 흐려서 판독 불가능
- 환자 식별 정보(PHI)가 노출됨
- 이미지가 깨지거나 로드 실패

**No (치명적 오류 아님)**:
- 이미지 품질이 다소 낮지만 판독 가능
- 약간의 아티팩트가 있으나 교육에 사용 가능
- 이미지와 텍스트가 일치함

#### 4.2.2 Image Anatomical Accuracy (해부학적 정확성) - 필수

**질문**: 이미지의 해부학적 구조와 라벨링이 얼마나 정확한가요?

| 점수 | 설명 |
| --- | --- |
| **1.0** | 완전히 정확 |
| **0.5** | 대체로 정확, 일부 불완전 |
| **0.0** | 부정확 |

**1.0점 (완전히 정확)**:
- 해부학적 구조가 모두 정확
- 라벨링 (화살표, 텍스트 등)이 정확한 위치
- 병변이나 소견이 정확히 표현됨

**0.5점 (대체로 정확, 일부 불완전)**:
- 주요 구조는 정확하나 일부 미세한 오류
- 라벨링이 약간 어긋나지만 이해 가능
- 병변 표현이 다소 불명확

**0.0점 (부정확)**:
- 해부학적 구조가 명백히 틀림
- 라벨링이 잘못된 위치를 가리킴
- 병변이 실제와 다르게 표현됨

#### 4.2.3 Image Quality (이미지 품질) - 필수

**질문**: 이미지의 전반적인 품질을 1-5점으로 평가하세요.

| 점수 | 설명 |
| --- | --- |
| **5점** | 매우 우수 - 선명하고 교육에 최적 |
| **4점** | 우수 - 대체로 좋음 |
| **3점** | 보통 - 사용 가능 |
| **2점** | 미흡 - 품질 개선 필요 |
| **1점** | 매우 미흡 - 거의 판독 불가 |

**평가 기준**:

**5점 (매우 우수)**:
- 이미지가 매우 선명하고 고해상도
- 중요 구조와 병변이 명확히 보임
- 노이즈나 아티팩트 없음
- 교육용으로 최적의 품질

**4점 (우수)**:
- 이미지가 선명하고 판독에 문제없음
- 약간의 노이즈는 있으나 큰 지장 없음
- 교육에 충분히 사용 가능

**3점 (보통)**:
- 평균적인 이미지 품질
- 판독 가능하나 일부 불명확
- 약간의 아티팩트나 흐림 존재

**2점 (미흡)**:
- 이미지가 다소 흐리거나 저해상도
- 판독에 어려움 있음
- 상당한 노이즈나 아티팩트

**1점 (매우 미흡)**:
- 이미지가 매우 흐리거나 깨짐
- 판독이 거의 불가능
- 교육용으로 부적절

#### 4.2.4 Image-Text Consistency (이미지-텍스트 일치도) - 필수

**질문**: 이미지가 카드의 텍스트 내용과 얼마나 일치하나요?

| 점수 | 설명 |
| --- | --- |
| **1.0** | 완벽히 일치 |
| **0.5** | 대체로 일치, 일부 불일치 |
| **0.0** | 불일치 |

**1.0점 (완벽히 일치)**:
- 이미지가 질문과 정확히 일치
- 해설에서 언급한 소견이 이미지에 모두 표현됨
- 이미지와 텍스트가 서로를 완벽히 보완

**0.5점 (대체로 일치, 일부 불일치)**:
- 주요 내용은 일치하나 일부 미스매치
- 텍스트에서 언급한 소견 일부가 이미지에서 불명확
- 전반적으로 일치하나 세부사항 차이

**0.0점 (불일치)**:
- 이미지와 텍스트가 명백히 다름
- 완전히 다른 케이스의 이미지
- 텍스트에서 설명한 소견이 이미지에 없음
