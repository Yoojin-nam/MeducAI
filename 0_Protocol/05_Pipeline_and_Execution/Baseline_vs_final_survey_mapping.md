# MeducAI 설문 문항 구조 재정리

본 문서는 **기존 IRB 승인 설문지(V2.0)**를 기준으로,
- **BASELINE 설문에 유지할 문항**
- **FINAL(배포 이후 평가) 전용으로 이동할 문항**
- **새롭게 추가가 필요한 문항**
을 체계적으로 구분·정리한 매핑 문서이다.

본 정리는 **IRB amendment 최소화**, **변화량 분석 가능성 극대화**, **Reviewer_09 교수 자문(accuracy·efficiency·quality 강화)**를 동시에 만족하는 것을 목표로 한다.

---

## 1. BASELINE 설문에 유지할 문항

> **목적**: 개인별 baseline 특성 및 사전 상태 측정 (covariates)

| 기존 섹션 | 문항 범위 | 처리 | 근거 |
|---|---|---|---|
| A. 일반적 특성 | A1–A4 | 유지 | 인구통계·수련환경 보정 변수 |
| B. 학습도구 사용 현황 | B5 | 유지 | 기존 학습 전략 baseline 파악 |
| C. 인지 부하 | C1–C3 (전체) | 유지 | baseline cognitive load 기준점 |
| D. 자기효능감 | D1–D5 | 유지 | 변화량 분석 핵심 지표 |
| G. 행동·정서 요인 | G1–G5 | 유지 | 스트레스·수면 등 주요 공변량 |
| H. 자기평가 | H1–H2 | 선택 유지 | 탐색적 보조 분석 |

📌 **주의**
- 문항 내용은 그대로 유지하되, 시점 표현을
  - “시험 준비 기간 동안” → “현재 시점 기준”으로만 경미 조정

---

## 2. FINAL(배포 이후) 전용으로 이동할 문항

> **목적**: MeducAI 사용 경험에 대한 사후 평가

| 기존 섹션 | 문항 범위 | 처리 | 이유 |
|---|---|---|---|
| B. 학습도구 사용 현황 | B1–B4 | FINAL 전용 | 사용 여부·기간·빈도는 사후 회상이 더 정확 |
| E. 학습 만족도 | E1–E4 | FINAL 전용 | 경험 이후에만 의미 있음 |
| F. 기술 수용성(TAM) | F1–F5 전체 | FINAL 전용 | MeducAI 사용자군 평가 전용 |
| H. 자유 의견 | H3 | FINAL 전용 | 종합 피드백 수집 |

📌 **권장 조정**
- “시험 준비 기간 동안” → “MeducAI 사용 기간 동안”
- B 섹션은 FINAL에서 **회상용 summary**로만 사용

---

## 3. 새로 추가할 FINAL 전용 핵심 문항

> **목적**: 기존 설문으로 포착되지 않는 AI 교육 도구의 핵심 성능 평가

### 3.1 Technical Accuracy (신규)

| 영역 | 문항 예시 | 척도 |
|---|---|---|
| 사실 정확성 | 제공된 내용에서 명백한 오류를 경험한 적이 있다 | 1–5 Likert |
| 오개념 위험 | 그럴듯하지만 틀린 설명으로 혼란을 느낀 적이 있다 | 1–5 Likert |
| 오류 영향 | 오류가 학습에 부정적 영향을 주었다 | Yes/No + 서술 |

---

### 3.2 Educational Quality / Exam Utility (신규)

| 영역 | 문항 예시 | 척도 |
|---|---|---|
| 시험 적합성 | 전문의 시험 대비에 직접적으로 도움이 되었다 | 1–5 Likert |
| 핵심성 | 토픽의 핵심을 잘 짚었다 | 1–5 Likert |
| 불필요성 | 시험과 무관한 정보가 많았다 (역문항) | 1–5 Likert |

---

### 3.3 Efficiency (정량 측정 중심, 신규)

| 영역 | 문항 예시 | 응답 형식 |
|---|---|---|
| 실제 소요 시간 | MeducAI 사용 시 학습에 소요된 시간 | 분(min) |
| 비교 시간 | 동일 작업을 기존 방식으로 했을 경우 예상 시간 | 분(min) |
| 수정 비용 | 오류 수정·검증에 소요된 시간 | 분(min) |
| 체감 효율 | 시간 절감 효과를 느꼈다 | 1–5 Likert |

---

### 3.4 Trust / Calibration (신규)

| 영역 | 문항 예시 | 척도 |
|---|---|---|
| 신뢰 | MeducAI 산출물을 전반적으로 신뢰할 수 있었다 | 1–5 Likert |
| 맹목 수용 | 별도 검증 없이 그대로 암기한 경우가 있다 | 1–5 Likert (역) |
| 비판적 사용 | 오류 가능성을 염두에 두고 사용했다 | 1–5 Likert |

---

## 4. 요약: IRB-friendly 재구성 원칙

- **삭제 없음**: 기존 문항은 유지하되 위치·타이밍만 조정
- **Baseline = 개인 특성·상태**, **FINAL = MeducAI 평가**로 명확 분리
- 신규 문항은 모두 **MeducAI 특이 성능(accuracy·quality·efficiency)**에 한정

👉 본 문서는 **IRB amendment 시 변경 요약표**로 그대로 활용 가능함.
