## 🔍 문제 해결 가이드

### 404 에러 발생 시
1. API 키가 로테이션되었는지 확인
2. 파일이 새 키로 재업로드되었는지 확인
3. `.batch_tracking.json`의 `api_key_number`가 올바른지 확인

### 429 에러 발생 시
1. 사용 가능한 다른 API 키가 있는지 확인
2. `.env` 파일에 활성화된 키가 있는지 확인
3. API 키 할당량이 소진되었는지 확인

### 배치가 건너뛰어지는 경우
1. `.batch_tracking.json`에서 이미 제출된 배치인지 확인
2. 배치 상태가 `JOB_STATE_PENDING`, `JOB_STATE_RUNNING`, `JOB_STATE_SUCCEEDED` 중 하나인지 확인

### 파일 재업로드 실패 시
1. JSONL 파일이 존재하는지 확인
2. 파일 크기가 제한을 초과하지 않는지 확인
3. 네트워크 연결 상태 확인
