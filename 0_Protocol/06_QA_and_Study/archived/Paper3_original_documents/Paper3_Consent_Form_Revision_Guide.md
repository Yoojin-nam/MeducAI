# Paper 3 동의서 수정 가이드 (인센티브 제공 시점 변경)

**Status:** Reference (IRB Amendment Guide)  
**Version:** 1.0  
**Date:** 2026-01-15  
**Purpose:** 동의서에 명시된 인센티브 제공 시점을 변경할 때의 IRB 수정 가이드

---

## 현재 동의서 내용 분석

### 현재 명시된 내용

**"5. 손실에 대한 보상" 섹션:**
```
"전문의 시험 종료 후 시행되는 설문조사를 완료해 주신 
선생님께는 감사의 뜻으로 소정의 모바일 쿠폰(답례품)을 
제공합니다."
```

**의미:**
- 제공 시점: **설문조사 완료 후** (Follow-up 설문 완료 후)
- 조건: 설문조사 완료

---

## "미리 제공"의 의미

### 제안하신 변경사항

**현재 (동의서):**
- Follow-up 설문 완료 후 제공

**변경 제안:**
- 동의서 서명 + Baseline 설문 완료 직후 제공

---

## IRB 수정 필요 여부

### ✅ IRB 수정 필요

**이유:**
- 동의서에 명시된 내용과 다름
- "참여자에게 제공하는 인센티브"는 IRB 승인 사항
- 시점 변경은 IRB 수정(Amendment) 필요

### 수정 유형

**Minor Amendment (경미한 수정):**
- 인센티브 제공 시점 변경
- 가치나 종류 변경 없음
- Coercion 위험 감소 (더 윤리적)

**승인 가능성: 높음 (90-95%)**

---

## 권장 수정 방안

### 옵션 1: Baseline 설문 완료 후 제공 (권장 ⭐⭐⭐⭐)

**수정 내용:**
```
"5. 손실에 대한 보상

본 연구에 참여해 주시는 선생님께는 감사의 뜻으로 
소정의 모바일 쿠폰(답례품)을 제공합니다.

- 제공 시점: Baseline 설문 완료 직후
- 사례품: 모바일 쿠폰 (10,000원)
- 참여 여부와 무관: 연구 설명을 들으신 모든 분께 
  제공됩니다 (참여하지 않더라도 사례품을 받으실 수 
  있습니다)."
```

**장점:**
- Coercion 위험 낮음
- Selection bias 감소
- IRB 승인 용이

---

### 옵션 2: 두 단계 제공 (대안)

**수정 내용:**
```
"5. 손실에 대한 보상

본 연구에 참여해 주시는 선생님께는 감사의 뜻으로 
소정의 모바일 쿠폰(답례품)을 제공합니다.

- 1차 제공: Baseline 설문 완료 직후 (10,000원)
- 2차 제공: Follow-up 설문 완료 후 (추가 10,000원)
- 총액: 20,000원

참여하지 않더라도 Baseline 설문만 완료하시면 
1차 사례품을 받으실 수 있습니다."
```

**장점:**
- 참여 동기 유지
- 완료 인센티브

**단점:**
- 비용 증가
- Selection bias 가능

---

## IRB 수정 신청서 작성 가이드

### 수정 사유

```
"본 연구의 인센티브 제공 시점을 변경하고자 합니다.

**변경 사유:**
1. 윤리적 고려: Baseline 설문 완료 직후 제공으로 
   Coercion 위험 감소
2. Selection bias 감소: 참여 여부와 무관하게 제공하여 
   선택 편향 최소화
3. 참여율 향상: 초기 참여 동기 부여

**변경 내용:**
- 기존: Follow-up 설문 완료 후 제공
- 변경: Baseline 설문 완료 직후 제공 (또는 모든 사람에게)

**윤리적 영향:**
- Coercion 위험 감소 (더 안전)
- Selection bias 감소 (더 윤리적)
- 참여자 권리 보호 강화"
```

### 수정된 동의서 초안

**"5. 손실에 대한 보상" 섹션 수정:**

```
5. 손실에 대한 보상

본 연구에 참여해 주시는 선생님께는 감사의 뜻으로 
소정의 모바일 쿠폰(답례품)을 제공합니다.

- 제공 시점: Baseline 설문 완료 직후
- 사례품: 모바일 쿠폰 (10,000원)
- 제공 대상: 
  * 연구 설명을 들으신 모든 분께 제공됩니다
  * 참여 여부와 무관하게 동일한 사례품을 제공합니다
  * 참여하지 않더라도 사례품을 받으실 수 있습니다

※ 본 사례품은 연구 참여를 강요하기 위한 것이 아니며, 
  참여 여부는 전적으로 선생님의 자율적인 의사에 
  달려 있습니다.
```

---

## IRB 승인 가능성 평가

### 승인 가능성: 90-95% (높음)

**이유:**
1. **Coercion 위험 감소**
   - Baseline 설문 완료 후 제공은 Coercion 위험 낮음
   - "참여하지 않아도 제공" 명시로 더욱 안전

2. **윤리적 우수성**
   - 모든 사람에게 동등한 대우
   - Selection bias 감소

3. **Minor Amendment**
   - 인센티브 시점 변경만 (가치/종류 변경 없음)
   - 연구 설계 변경 없음

## 구현 절차

### 1. IRB 수정 신청

**필수 문서:**
- 수정 신청서
- 수정된 동의서 초안
- 수정 사유서

**제출 시점:**
- 연구 시작 전
- 또는 연구 진행 중이라면 즉시

### 2. 수정된 동의서 사용

**IRB 승인 후:**
- 승인된 수정 동의서 사용
- 기존 참여자에게는 재동의 불필요 (이미 참여 중인 경우)
- 신규 참여자부터 수정된 동의서 사용

### 3. 논문 보고

**Methods 섹션:**
```
"Participants received a small token of appreciation 
(mobile coupon, 10,000 KRW) immediately upon completion 
of the baseline survey. This incentive was provided to 
all potential participants regardless of their decision 
to continue participation, as approved by the IRB 
(Amendment #XX, Date: YYYY-MM-DD)."
```

## 주의사항

### 1. IRB 승인 전 제공 금지

**절대 하지 말 것:**
- IRB 승인 전에 수정된 방식으로 제공
- 동의서에 명시된 내용과 다르게 제공

**이유:**
- IRB 위반
- 연구 승인 취소 가능

### 2. 기존 참여자 처리

**상황:**
- 이미 동의서 서명한 참여자가 있는 경우

**처리:**
- IRB 승인 후 신규 참여자부터 적용
- 기존 참여자는 기존 약속대로 (Follow-up 후 제공)
- 또는 IRB 승인 후 재동의 요청 (선택)
