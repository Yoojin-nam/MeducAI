## 우선순위별 개선점

### P0: 치명적 오류 (즉시 수정 필요)

**설명**: 학습자에게 잘못된 정보를 전달하거나 오인을 유발할 수 있는 치명적 오류

#### 주요 문제점

1. **이미지-지문 불일치**
   - 지문은 CT 영상인데 제시된 사진은 US/초음파
   - 지문은 MRI 소견인데 제시된 사진은 CT
   - 지문과 영상 소견이 정반대

2. **해부학적 오류**
   - 림프절 위치가 잘못 표시됨
   - 해부학적 랜드마크 부족
   - 모식도에서 방향/위치 오류

3. **진단 용어 오류**
   - Entity 이름 오류
   - 진단 용어와 영상 소견 불일치

#### 구체적 사례

- **Arm A, group_07**: "Malignant lymphoma 같은 경우 제시된 CT 영상이 오히려 대부분 necrotic LN enlargement로 묘사되어 있어 오류"
- **Arm A, group_14**: "지문은 경계가 명확한 원형 결절이라고 하였으나 CT 영상은 spiculated margin을 보이는 결절로 서로 mismatch"
- **Arm B, group_04**: "기형종 Q2 MCQ 문제에서 MRI 소견을 묻는 지문의 제시 사진이 CT로 되어 있음"
- **Arm C, group_03**: "Q1SET의 폐혈류이상 Q1과 Q2에 제시된 사진이 똑같은데, 문제나 답은 정반대 양상을 고르게 되어 있음"

#### 개선 방안

1. **이미지 검증 프로세스 강화**
   - 지문과 이미지 매칭 자동 검증
   - 모달리티 일치 확인 (CT vs MRI vs US)
   - 영상 소견과 지문 내용 일치 확인

2. **해부학적 정확성 검증**
   - 해부학 전문가 리뷰
   - 해부학적 랜드마크 명시
   - 모식도 검증 프로세스

3. **용어 및 Entity 검증**
   - 의학 용어 사전 검증
   - Entity 이름과 내용 일치 확인

### P1: 이미지/인포그래픽 품질 개선 (높은 우선순위)

**설명**: 이미지와 인포그래픽의 정확성, 적절성, 교육적 효과 개선

#### 주요 문제점

1. **영상 소견 부정확**
   - 실제 소견과 다른 영상 사용
   - 특징적 소견이 명확하지 않음
   - 감별 포인트가 드러나지 않음

2. **모식도/인포그래픽 부적절**
   - 해부학적 정확성 부족
   - 방향/위치 표시 오류
   - 핵심 정보 누락

3. **이미지 선택 부적절**
   - 교육 목표와 부합하지 않는 이미지
   - 작위적이거나 인위적인 이미지
   - 학습 효과가 낮은 이미지

#### 구체적 사례

- **Arm A, group_02**: "compression과 view의 방향이 모식도에서 잘 못 표시되어 있음"
- **Arm A, group_05**: "May-thurner syndrome의 인포그래픽이 부적절. DVT의 초음파 소견도 Probe종류나 영상소견이 부적절함"
- **Arm A, group_09**: "인포그래픽에서 tumor 위치 (GCT, SBC), 모양 (enchondroma, SBC, LCH)이 틀림"
- **Arm B, group_07**: "림프절 위치가 이상하게 그려짐, 영상소견이 명확하게 표현되지 않음, 소견이 다 비슷함. 그림만 봐서는 감별이 안 됨"
- **Arm E, group_05**: "인터벤션이미지 설명이 아님. 메이터너같은 경우 그림이 아예 틀림"

#### 개선 방안

1. **이미지 선택 가이드라인 수립**
   - 교육 목표와의 부합성 확인
   - 특징적 소견이 명확한 이미지 우선 선택
   - 감별 포인트가 드러나는 이미지 선택

2. **인포그래픽 검증 프로세스**
   - 해부학적 정확성 검증
   - 방향/위치 표시 정확성 확인
   - 핵심 정보 포함 여부 확인

3. **전문가 리뷰 강화**
   - 각 subspecialty 전문가 리뷰
   - 영상 소견 정확성 검증
   - 교육적 효과 평가

### P2: 정확성 및 일관성 개선 (중간 우선순위)

**설명**: 지문-답-해설 간 일관성 및 정확성 개선

#### 주요 문제점

1. **답과 해설 불일치**
   - 답은 맞지만 해설과 불일치
   - 해설이 답을 제대로 설명하지 못함

2. **중복 정답 가능성**
   - 여러 답이 가능한 경우
   - 감별이 어려운 경우

3. **용어 및 개념 정확성**
   - 진단 용어 vs 영상 소견 용어 혼동
   - Entity 이름 오류

#### 구체적 사례

- **Arm A, group_04**: "생리적 낭종의 크기가 5cm 초과하는 경우 추적관찰을 고려한다"를 적절한 권고사항으로 뽑았는데, 답도 틀렸고 해설과도 일치하지 않음
- **Arm A, group_07**: "결핵성 림프절염 Q2 Type: MCQ 문항은 중복 정답의 가능성이 있음. 단순히 intranodal cystic change + multiple + matted appearance만으로는 TB lymphadenitis와 metastatic LN를 감별하기가 쉽지 않아 보입니다"
- **Arm A, group_14**: "SPN이라는 것은 진단이라기 보다는 영상 소견으로 보는 게 더 맞아 보입니다. 또한 SPN이라고 하는 것은 중립적인 기술 용어이지만 제시된 사진 자체는 악성의 가능성이 높은 사진으로 혼란을 줄 수 있습니다"
- **Arm C, group_01**: "합병증성 담낭염이라는 entity 자체가 잘못된 듯 합니다. 답이 GB perforation이어야 맞지 않을까 싶습니다"

#### 개선 방안

1. **답-해설 일관성 검증**
   - 자동 검증 프로세스
   - 해설이 답을 명확히 설명하는지 확인

2. **중복 정답 방지**
   - 감별 진단 고려
   - 명확한 감별 포인트 제시
   - 전문가 리뷰를 통한 검증

3. **용어 정확성 개선**
   - 진단 용어 vs 영상 소견 용어 구분
   - Entity 이름 정확성 확인
   - 의학 용어 사전 활용

#### 개선 방안

   - 의학 용어 사전 활용
   - 한글명 정확성 확인
   - 오타 검사 자동화

2. **가독성 개선**
   - Font 렌더링 문제 해결
   - 표기 일관성 확인
   - 레이아웃 최적화

### P4: 구조 및 형식 개선 (낮은 우선순위)

**설명**: 테이블/인포그래픽 구조 및 형식 개선

#### 주요 문제점

1. **테이블 구조**
   - 테이블이 너무 복잡함
   - 핵심 내용이 산발적
   - 정제 필요

2. **정보 누락**
   - 표가 다 안 채워져 있음
   - 필수 정보 누락

#### 구체적 사례

- **Arm E, group_03**: "table이 너무 복잡해서 보기가 어려움"
- **Arm F, group_01**: "Master table의 구성 및 내용의 정확도가 떨어짐. 중요한 내용이 잘 포함되어 있는 것 같긴 하나 이를 이와 같은 한 table로 묶는 것이 적절한지에 대한 의문이 듭니다"
- **Arm D, group_13**: "표가 다 안채워져있음 (세팅/파라미터, 아티팩트 열)"
- **Arm F, group_13**: "Table에 핵심적인 내용이 들어가있긴 하지만 너무 산발적인 주제들에 대해서 정리가 되어 있어 정제가 필요해보입니다"

#### 개선 방안

1. **테이블 구조 최적화**
   - 복잡도 감소
   - 핵심 내용 중심으로 재구성
   - 정보 계층화

2. **정보 완성도 개선**
   - 필수 정보 누락 방지
   - 자동 검증 프로세스
   - 완성도 체크리스트
