# Google Form 설정 가이드 (상세)

**Date:** 2025-12-20  
**Status:** 실행 가이드  
**Purpose:** Google Form의 이메일 인증 및 중간 저장 설정 방법

---

## 1. 기본 설정 접근

1. **Google Form 편집 화면**에서
2. 오른쪽 상단의 **설정 아이콘 (톱니바퀴)** 클릭
3. 또는 **설정** 메뉴 클릭

---

## 2. 이메일 인증 설정

### 2.1 이메일 수집 활성화

**경로:** 설정 > 일반 > 응답

1. **"이메일 주소 수집"** 옵션 찾기
2. ✅ **체크박스 활성화**
3. (선택) "이메일 주소 확인" 옵션도 활성화 가능

**효과:**
- 응답자가 Google 계정으로 로그인해야 응답 가능
- 응답자의 이메일 주소가 자동으로 수집됨
- 중복 응답 방지에 도움

### 2.2 1인당 1회 응답 제한

**경로:** 설정 > 일반 > 응답

1. **"1인당 1회만 응답"** 옵션 찾기
2. ✅ **체크박스 활성화**

**효과:**
- 동일 Google 계정으로는 1회만 응답 가능
- 중복 응답 완전 차단

**주의:**
- 이메일 수집과 함께 사용해야 효과적
- 이메일 수집 없이 사용하면 쿠키 기반으로만 제한됨

## 3. 중간 저장 및 이어서 응답

### 3.1 응답 수정 허용

**경로:** 설정 > 일반 > 응답

1. **"응답 수정 허용"** 옵션 찾기
2. ✅ **체크박스 활성화**

**효과:**
- 응답 제출 후에도 수정 가능
- 하지만 **중간 저장 기능은 아님**

### 3.2 자동 저장 (Draft 기능)

Google Form은 **자동으로 초안을 저장**합니다.

**작동 방식:**
1. 응답자가 Form을 작성하는 동안
2. Google 계정으로 로그인되어 있으면
3. **자동으로 초안이 저장됨**
4. 나중에 같은 Google 계정으로 접속하면
5. **"초안 계속 작성"** 링크가 표시됨

**활성화 조건:**
- ✅ Google 계정 로그인 필수
- ✅ 이메일 수집 활성화
- ✅ 자동으로 활성화됨 (별도 설정 불필요)

### 3.3 초안 복원 방법 (응답자용)

**응답자가 초안을 이어서 작성하는 방법:**

1. **같은 Google 계정으로 Form 링크 접속**
2. **"초안 계속 작성"** 메시지 확인
3. 클릭하여 이전에 작성한 내용 복원
4. 이어서 작성 후 제출

**주의:**
- 다른 브라우저나 시크릿 모드에서는 초안이 복원되지 않을 수 있음
- 같은 Google 계정, 같은 브라우저에서 접속해야 함

## 4. 전체 설정 체크리스트

### 4.1 필수 설정

- [ ] **이메일 주소 수집**: ✅ 활성화
- [ ] **1인당 1회만 응답**: ✅ 활성화
- [ ] **응답 수정 허용**: ✅ 활성화 (중간 저장 후 이어서 작성 가능)

### 4.2 선택 설정

- [ ] **이메일 주소 확인**: (선택) 이메일 주소 형식 검증
- [ ] **응답 후 링크 표시**: ❌ 비활성화 (블라인딩 유지)
- [ ] **응답 요약 보기**: (선택) 평가자에게는 불필요

## 6. 실제 설정 단계별 가이드

### Step 1: Form 생성 후 기본 설정

1. **Google Forms 접속**: https://forms.google.com
2. **새 양식 만들기**
3. **제목 및 설명 입력**
4. **설정 아이콘 (⚙️) 클릭**

### Step 2: 이메일 인증 설정

1. **"일반" 탭** 선택
2. **"응답" 섹션**으로 스크롤
3. **"이메일 주소 수집"** 체크박스 ✅
4. **"1인당 1회만 응답"** 체크박스 ✅

### Step 3: 중간 저장 설정

1. **"응답 수정 허용"** 체크박스 ✅
2. **"응답 후 링크 표시"** 체크박스 ❌ (체크 해제)

### Step 4: 확인 및 저장

1. 설정 창 닫기 (자동 저장됨)
2. Form 테스트:
   - 본인 이메일로 테스트 응답 제출
   - 중간에 중단 후 다시 접속하여 초안 복원 확인

## 7. 응답자 안내 (이메일/Form 설명에 포함)

### 7.1 중간 저장 안내 문구

Form 설명 또는 이메일 본문에 다음 내용을 포함하세요:

```
[중간 저장 안내]

- Google 계정으로 로그인하여 응답하시면 자동으로 초안이 저장됩니다.
- 중간에 중단하셔도 나중에 같은 Google 계정으로 접속하시면 
  "초안 계속 작성" 링크가 표시됩니다.
- 같은 브라우저, 같은 Google 계정에서 접속하시면 
  이전 작성 내용이 자동으로 복원됩니다.
- 모든 항목을 완료한 후 "제출" 버튼을 클릭해 주세요.
```

### 7.2 Form 설명란에 추가

Form 상단 설명란에 다음을 추가할 수 있습니다:

```
※ Google 계정으로 로그인하여 응답하시면 자동으로 초안이 저장됩니다.
  중간에 중단하셔도 나중에 같은 계정으로 접속하시면 이어서 작성하실 수 있습니다.
```

## 8. 테스트 방법

### 8.1 이메일 인증 테스트

1. **시크릿/프라이빗 브라우저**에서 Form 링크 접속
2. **로그인 없이 응답 시도**
3. ✅ **로그인 요구 메시지 표시 확인**
4. **Google 계정으로 로그인**
5. ✅ **응답 가능 확인**

### 8.2 중복 응답 방지 테스트

1. **첫 번째 응답 제출**
2. **같은 계정으로 다시 Form 접속**
3. ✅ **"이미 응답했습니다" 메시지 확인**
4. ✅ **응답 수정 허용 시 "응답 수정" 링크 표시 확인**

### 8.3 중간 저장 테스트

1. **Form 작성 시작** (몇 개 항목만 입력)
2. **브라우저 탭 닫기** (제출하지 않음)
3. **같은 Google 계정, 같은 브라우저로 다시 접속**
4. ✅ **"초안 계속 작성" 메시지 확인**
5. ✅ **이전 입력 내용 복원 확인**
6. **이어서 작성 후 제출**

## 9. 문제 해결

### Q1: 초안이 저장되지 않아요

**원인:**
- Google 계정으로 로그인하지 않음
- 시크릿/프라이빗 모드 사용
- 다른 브라우저 사용

**해결:**
- Google 계정으로 로그인 확인
- 일반 브라우저 모드 사용
- 같은 브라우저에서 접속

### Q2: 중복 응답이 가능해요

**원인:**
- "이메일 주소 수집" 미활성화
- "1인당 1회만 응답" 미활성화

**해결:**
- 두 설정 모두 활성화 확인

### Q3: 응답 수정이 안 돼요

**원인:**
- "응답 수정 허용" 미활성화

**해결:**
- 설정에서 "응답 수정 허용" 활성화

### Q4: 초안 복원이 안 돼요

**원인:**
- 다른 브라우저/기기 사용
- 다른 Google 계정 사용
- 초안 저장 시간 경과 (일정 시간 후 자동 삭제 가능)

**해결:**
- 같은 브라우저, 같은 계정 사용 안내
- 가능한 한 빠른 시간 내 완료 권장

## 10. Google Apps Script로 설정 확인

Google Apps Script에서 Form 설정을 확인하는 방법:

```javascript
function checkFormSettings() {
  const form = FormApp.getActiveForm();
  
  // 이메일 수집 확인
  const collectsEmail = form.collectsEmail();
  Logger.log('이메일 수집: ' + collectsEmail);
  
  // 1인당 1회 응답 제한 확인
  const limitOneResponsePerUser = form.getLimitOneResponsePerUser();
  Logger.log('1인당 1회 응답 제한: ' + limitOneResponsePerUser);
  
  // 응답 수정 허용 확인
  const allowsResponseEdits = form.allowsResponseEdits();
  Logger.log('응답 수정 허용: ' + allowsResponseEdits);
  
  // 응답 후 링크 표시 확인
  const showsLinkToRespondAgain = form.getShowsLinkToRespondAgain();
  Logger.log('응답 후 링크 표시: ' + showsLinkToRespondAgain);
}
```

## 12. Form 생성 스크립트에서 설정

`create_google_form_qa.js` 스크립트에서 이미 다음 설정이 포함되어 있습니다:

```javascript
form.setCollectEmail(true); // 이메일 수집
form.setLimitOneResponsePerUser(true); // 1인당 1회 응답 제한
form.setAllowResponseEdits(true); // 응답 수정 허용
form.setShowLinkToRespondAgain(false); // 응답 후 링크 표시 안 함
```

**스크립트를 사용하면 자동으로 설정됩니다!**
